# Diagrams

Educational diagrams for the WuBu Nesting project, created as SVG files.

## Available Diagrams

| Diagram | Description |
|---------|-------------|
| `wubu-nesting-architecture.svg` | The nested hyperbolic sphere architecture — Russian doll concept with all 9 components labeled |
| `hamilton-encoder-pipeline.svg` | End-to-end pipeline: RGB input → 2×2 pooling → quaternion encoding → BSP tree → fused attention |
| `research-timeline.svg` | Complete research arc from August 2025 to April 2026 across 5 phases |

## How to View

All diagrams are standalone SVG files. Open them in any browser or SVG viewer.

## How to Regenerate

These SVGs were hand-crafted for clarity. To modify:
1. Open the SVG in any text editor
2. Adjust coordinates, colors, or text
3. Save and reopen in browser

For mermaid diagrams (used in markdown), see:
- `THEORY/03-wubu-nesting-paper.md` — the full architecture mermaid diagram
- Render with any mermaid CLI or at https://mermaid.live

# GAN-VAE Hybrid Diffusion

This directory documents the GAAD (Geometric Adversarial Autoencoder Diffusion) family of models — hybrids of GANs, VAEs, and diffusion.

The actual code lives in `draftPY/`:
- `WuBuGAADHybridGen_v0.1.py` through `v0.3.py`
- `WuBuNestDiffusion_v0.05_GAAD_MotionWuBu_PaperCopy.py`
- `WuBuNestDiffusion_v0.10.0_GAAD_RegionalLatent.py`
- `WuBuNestDiffusion_v0.10.1_OpticalFlow.py`

Key features:
- Regional latent processing with optical flow
- Motion vectors and temporal attention
- Geometric adversarial training
